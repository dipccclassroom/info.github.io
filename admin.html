🚀 Publish Changes

How publishing works: Enter your GitHub Personal Access Token. The dashboard calls the GitHub API to update data.js, then GitHub Actions auto-deploys (30-60 sec).

Create token: GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens → Contents: Read & Write for this repo.

Your token can be kept in this browser session only, then cleared automatically when the session ends.
