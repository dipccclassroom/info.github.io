🚀 Publish Changes

How publishing works: Enter your GitHub Personal Access Token below. When you click "Publish", the dashboard will call the GitHub API to update index.html in your repository. GitHub Actions will then auto-deploy the changes (usually within 30 seconds).

To create a token: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Grant Contents: Read & Write to this repo.

Your token is never stored — it lives only in memory for this session.
